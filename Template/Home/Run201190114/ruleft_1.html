【幸运飞艇游戏介绍】

幸运飞艇：是马耳他共和国瓦莱塔福利联合委员会独家发行的一款高频彩，源于F1赛艇的彩票游戏；
幸运飞艇以「1~10」十个号码作为开奖依据，完全公平公正公开，开奖透明，无法作弊！

【相关资料】

【开奖时间】
幸运飞艇为每天下午13:05~晚上04:05止；
每五分钟开奖一期，每次投注时间约220秒，等待开奖时间约为1分钟，与官网完全同步。

【玩法】

【1~10名猜大小单双】

第一名～第十名车号：开出之号码大于或等于6为大，小于或等于5为小。开出的号码偶数为双，号码奇数为单。

■赔率：含本{:C('ft_chehao')}倍
■限额：最低：{:C('ft_min_point')}
最高：每个号码最高{:C('ft_xz_max')['dxds']}
■格式：名次大小单双金额
■格式：大小单双金额
例：12345双100 = 1~5车道买双各$100 = 总$500 不填写名次，如大100，则默认为1大100

【1~10名猜车号】

每一个车号为一投注组合，开奖结果「投注车号」对应所投「车道」视为中奖，其余情形视为不中奖。

■赔率：含本{:C('ft_chehao')}倍
■限额：{:C('ft_min_point')}-{:C('ft_xz_max')['chehao']}
■格式：名次/号码/金额
例：12345/89/20 = 1~5车道的8号、9号各买$20 = 总$200 例：1357/890/20 = 1.3.5.7车道的8号、9号、10号各买$20 = 总$240

【1~10名猜组合】

竞猜内容为「大单」「小双」「小单」「大双」，共4种。

■赔率：
「大单」、「小双」含本{:C('ft_zuhe_1')}倍， 「小单」、「大双」含本{:C('ft_zuhe_2')}倍。 ■限额：{:C('ft_min_point')}-{:C('ft_xz_max')['zuhe']}
■格式：名次/组合/金额
例：890/大单/50 = 8.9.10车道大单各买$50 = 总$150

【1~5名猜龙虎】

(1)第一名vs第十名，(2)第二名vs第九名，(3)第三名vs第八名，(4)第四名vs第七名，(5)第五名vs第六名，前比后大为龙，反之为虎。

■赔率：含本{:C('ft_lh')}倍
■限额：{:C('ft_min_point')}-{:C('ft_xz_max')['lh']}
■格式：名次/号码/金额
例：123/龙/100 = 1~3车道买龙各$100=总$300

【冠亚猜庄闲】

冠军车号「大于」亚军车号为庄，反之为闲。

■赔率：含本{:C('ft_zx')}倍
■限额：{:C('ft_min_point')}-{:C('ft_xz_max')['zx']}
■格式：庄闲/金额
例：庄/200 = 冠军大于亚军即中奖

【猜冠亚号码】

猜冠军及亚军车号，每次竞猜2个号码，顺序不限。

■赔率：含本{:C('ft_gy')}倍
■限额：{:C('ft_min_point')}-{:C('ft_xz_max')['gy']}
■格式：组/号码/金额
例：组/5-6/50 = 5号.6号车在冠亚军(顺序不限) = 总$50 例：组/1-9.3-7/100 = 1.9号车或3.7号车在冠亚军(顺序不限) = 总$200

【冠亚和值(特码)猜大小单双】

冠军车号+亚军车号 = 冠亚和值 = 特码 = 数字3~19

冠亚和值大于或等于12为大，小于或等于11为小。开出的号码偶数为双，号码奇数为单。

■赔率：
「大」、「双」、「小」、「单」含本{:C('ft_tema')}倍。 「大」、「双」的赔率为{:C('ft_tema_1')}倍。 「小」、「单」的赔率为{:C('ft_tema_2')}倍。 ■限额：{:C('ft_min_point')}-{:C('ft_xz_max')['tema']}
■格式：和(特)/大小单双/金额
例：和双100 = 「冠亚和」的双$100 例：和大100 = 「冠亚和」的大$100

【冠亚和值(特码)猜数字】

「冠亚和值」为「特码」可能出现的结果为3~19，竞猜中对应「冠亚和值」数字的视为中奖，其余视为不中奖。

■赔率：
3.4.18.19，含本{:C('ft_tema_sz_1')}倍，限额20-1,000
5.6.16.17，含本{:C('ft_tema_sz_2')}倍，限额20-2,000
7.8.14.15，含本{:C('ft_tema_sz_3')}倍，限额20-3,000
9.10.12.13，含本{:C('ft_tema_sz_4')}倍，限额20-4,000
11，含本8倍，限额20-5,000 ■限额：{:C('ft_min_point')}-{:C('ft_xz_max')['tema_sz']}
■格式：和(特)/数字/金额
例：和567/100 = 竞猜「冠亚和」的值为5或6或7各$100 = 总$300

【冠亚和值(特码)猜区段】

特码3~7为A段，8~14为B段，15~19为C段。

■赔率：
「A」、「C」含本{:C('ft_tema_qd_1')}倍。
「B」含本{:C('ft_tema_qd_2')}倍。 ■竞猜限额：{:C('ft_min_point')}-{:C('ft_xz_max')['tema_qd']}
■格式：ABC/金额
例：BC/100 = 投注BC段位各100＝总$200

每期下注：总额10万封顶
